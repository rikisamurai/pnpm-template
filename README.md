# PNPM Workspace Template

- [Workspace | pnpm](https://pnpm.io/workspaces)
- [pnpm-workspace.yaml | pnpm](https://pnpm.io/pnpm-workspace_yaml)

## Development

- node ≥ 20.18.0
- pnpm ≥ 9.12.2

### Setup

```bash
# install and setup
pnpm run setup
```

## Daily Commands

```bash
# install
pnpm install
# filter
pnpm --filter vite-react dev
```

> [Filtering | pnpm](https://pnpm.io/filtering)
